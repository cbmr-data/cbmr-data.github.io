``Chains not found`` when connecting to sftp.ku.dk
==================================================

If you try and fail to log in to sftp.ku.dk too many times in a short
period of time, then your account may get locked out. In that case,
please create a ticket using KU-IT serviceportal_.

File uploads using MobaXterm never start
========================================

Please make sure that your session is configured to use the ``SCP
(enhanced speed)`` browser type. See step 4 in the
:ref:`s_configure_mobaxterm` section.
